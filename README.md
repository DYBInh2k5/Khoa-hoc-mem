# IT-Support

1. Tổng quan nghề IT Support
IT Support hỗ trợ người dùng và doanh nghiệp giải quyết sự cố kỹ thuật, đảm bảo hệ thống CNTT hoạt động ổn định.​

Công việc chính: khắc phục sự cố, bảo trì máy, hỗ trợ khách hàng, phối hợp với các bộ phận khác.​

2. Kiến thức về phần cứng và dữ liệu
Nhận diện linh kiện cơ bản: CPU, RAM, ổ cứng, mainboard, nguồn, thiết bị ngoại vi.​

Khái niệm bit, byte và các đơn vị KB, MB, GB, TB trong lưu trữ và truyền dữ liệu.​

3. Mạng máy tính cơ bản
Các loại mạng: LAN, WAN, Wi‑Fi; ứng dụng trong gia đình và văn phòng nhỏ.​

Thiết bị mạng: modem, router, switch, access point và vai trò của chúng.​

Các thao tác cơ bản: kiểm tra kết nối, kiểm tra IP, gateway, DNS, tốc độ mạng.​

4. Hệ điều hành và sử dụng máy tính
Khái niệm hệ điều hành và các hệ điều hành phổ biến: Windows, Linux, macOS.​

Các thao tác chính: quản lý file/thư mục, cài/gỡ phần mềm, cập nhật hệ thống, driver.​

Làm quen dòng lệnh (CLI), đặc biệt trên Linux để quản lý file, người dùng, gói phần mềm.​

5. Quản trị hệ thống cơ bản
Phân biệt máy chủ, máy trạm, dịch vụ chia sẻ file, in, email, web trong tổ chức.​

Quản lý tài khoản và phân quyền người dùng (admin, user thường), nguyên tắc “quyền tối thiểu”.​

Chia sẻ file và máy in qua LAN, theo dõi hiệu năng hệ thống (CPU, RAM, ổ đĩa).​

6. Bảo mật và xử lý rủi ro
Nguy cơ bảo mật: virus, malware, lộ mật khẩu, truy cập trái phép, mất dữ liệu.​

Nguyên tắc: mật khẩu mạnh, cập nhật phần mềm, phân quyền hợp lý, sao lưu dữ liệu.​

Quy trình xử lý sự cố: xác định vấn đề, kiểm tra, khắc phục, phục hồi, phòng ngừa tái diễn.​

7. Dự án cuối môn: Hệ thống văn phòng nhỏ
Thiết kế sơ đồ mạng: 2–3 máy tính, 1 máy in, 1 router/modem, kết nối dây hoặc Wi‑Fi.​

Cài hệ điều hành, tạo tài khoản, phân quyền, chia sẻ file/máy in, kiểm tra bảo mật cơ bản.​

Viết báo cáo mô tả quá trình triển khai và đề xuất cải thiện bảo mật, vận hành lâu dài.​
